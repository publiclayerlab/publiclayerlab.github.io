title: "Accessibility"
description: "Accessibility information for the Public Layer Lab website."
---

Public Layer Lab aims to make this website readable, accessible, and usable.

The site is currently a small static website under active development. It is built with simple pages, plain navigation, readable typography, and minimal scripts.

We do not currently claim formal WCAG compliance. As the website develops, accessibility will be reviewed and improved.

If you encounter an accessibility barrier, unclear structure, unreadable text, keyboard-navigation problem, or other issue, please contact:

publiclayerlab@gmail.com
